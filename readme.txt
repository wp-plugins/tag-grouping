Create logical and meaningful groups of tags. Update old posts by updating your tag group. Add tags in bulk with 1 selection.

== Description ==

This plug-in allows administrators the ability to create groups of tags that they commonly use. These groups can be assigned to old/new posts.
If, later, you find that you want to add some new tags to your group, all of the posts assigned to the group will receive the new tags.

== Installation ==

1. Upload the contents of the zip file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create your Groups.
4. Add them to your posts.
5. Send any feedback or suggestions to mike@croutonsoflife.com
